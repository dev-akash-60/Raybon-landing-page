# Raybon Landing Page

A modern animated landing page built using HTML, CSS, JavaScript, GSAP, and Shery.js.

---

## 🚀 Features

- Gooey image distortion effect
- Smooth GSAP text animations
- Interactive hero section
- Responsive modern UI
- Clean and minimal design
- Infinite text slider animation

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP
- Shery.js
- Three.js

---

## 📁 Folder Structure

```bash
raybon-landing-page/
│
├── index.html
├── style.css
├── script.js
└── assets/
    └── images/
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/raybon-landing-page.git
```

Move into the project folder:

```bash
cd raybon-landing-page
```

Run with Live Server or open `index.html` directly in your browser.

---

## 🔗 CDN Links

Add these before the closing `</body>` tag:

```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script>

<script src="https://unpkg.com/sheryjs/dist/Shery.js"></script>
```
